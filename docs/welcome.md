### Authentication

The breeze payment gateway uses API keys to authenticate requests. You can view and manage your API keys in **Project Setting**.

:::warning

#### Your API keys carry many privileges, so be sure keep them secure!

:::

### Making your first request

Trigger a payment session by sending a POST request to the <Highlight>/api</Highlight> endpoint.
